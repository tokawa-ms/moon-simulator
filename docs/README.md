# 🌙 月の満ち欠けシミュレーター

小学校理科教材向けの3D月相可視化アプリケーション

## 📖 概要

このアプリケーションは、太陽・地球・月の位置関係と月の満ち欠けの仕組みを視覚的に学習できる教育ツールです。Three.jsを使用した3D表現により、宇宙空間での天体の配置や、異なる視点からの月の見え方を直感的に理解できます。

## ✨ 主な機能

- 🌌 **3D空間ビュー**: 太陽・地球・月の位置関係を宇宙から観察
- 🌍 **地球視点**: 地球から見た月の満ち欠け
- 🌕 **月視点**: 月から見た地球の見え方
- 🗾 **日本視点**: 日本から観測した月の形と名称
- ⏱️ **タイムコントロール**: 月の公転と地球の自転をスライダーで操作
- ▶️ **自動アニメーション**: 月の公転を自動的に再生
- 🖱️ **インタラクティブ操作**: マウスで視点を自由に変更

## 🚀 クイックスタート

### 必要な環境

- モダンなWebブラウザ（Chrome 90+、Firefox 88+、Safari 14+、Edge 90+）
- インターネット接続（初回読み込み時にCDNからライブラリを取得）

### 起動方法

1. リポジトリをダウンロードまたはクローン
2. `src/index.html` をブラウザで開く
3. すぐに使用開始！（ビルド・インストール不要）

```bash
# Gitでクローンする場合
git clone https://github.com/tokawa-ms/moon-simulator.git
cd moon-simulator

# ブラウザで開く（例: macOS）
open src/index.html

# ブラウザで開く（例: Linux）
xdg-open src/index.html

# ブラウザで開く（例: Windows）
start src/index.html
```

## 📚 使い方

### 基本操作

1. **メインビュー（3D空間）**
   - マウスドラッグ: 視点を回転
   - マウスホイール: ズームイン/アウト

2. **月の公転スライダー**
   - 0〜29.5日の範囲で月の位置を調整

3. **地球の自転スライダー**
   - 0〜24時の範囲で時刻を調整

4. **ボタン**
   - 再生: 自動アニメーション開始/停止
   - リセット: 初期状態に戻す

### 学習のポイント

- 月の満ち欠けは、太陽・地球・月の位置関係によって決まる
- 月自体は光っておらず、太陽の光を反射している
- 月の公転周期は約29.5日（朔望月）
- 視点によって天体の見え方が大きく変わる

詳しくは [使い方ガイド](docs/user-guide.md) をご覧ください。

## 🎓 教育的価値

このシミュレーターは以下の学習目標をサポートします:

1. **天体の動き**: 月の公転、地球の自転の理解
2. **月の満ち欠けの原理**: 太陽光の反射による見え方の変化
3. **空間認識**: 3次元空間での天体配置の把握
4. **視点の転換**: 異なる視点からの観測体験
5. **周期性**: 自然現象の周期的な変化の理解
